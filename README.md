# dotfiles
These are my `.config` files for my Linux computer. It features...
- using hyprland, a beautiful hyper-configurable tiling window manager.
- using kitty, a pretty terminal that is also fast and non-distracting.
- using neovim, a lightweight but powerful code editor.
- using waybar, a simple bar featuring time, workspaces, window name, CPU temperature, RAM usage, CPU usage, and volume.
- using zathura, a minimal vim-like PDF viewer that reloads automatically
